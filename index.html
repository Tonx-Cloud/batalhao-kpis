Referencial Teórico

Fórmulas e lógica gerencial segundo Harvard / MIT Sloan / London Business School.

1. Indicadores Centrais (Core KPIs)

  • NPS = %Promotores − %Detratores Reichheld · Harvard
  • CSAT = (satisfeitos / total de respostas) × 100
  • Retenção = ((Finais − Novos) / Iniciais) × 100
  • Churn = (Perdidos / Totais) × 100
  • LTV = (Receita média × Margem) / Churn

2. Indicadores Operacionais

  • Tempo médio = Σ tempo / nº atendimentos
  • FCR = resolvidos no 1º contato / total de problemas
  • Conversão = (vendas / atendimentos) × 100
  • Ticket médio = receita / nº vendas

3. Indicadores Avançados

  • CAC = marketing / clientes adquiridos
  • LTV/CAC ≥ 3 (regra Harvard/MIT)
  • Produtividade = receita / nº funcionários

4. Diferencial Pedagógico

  • Simulador de decisões gerenciais
  • Comparação entre empresas / cenários
  • Benchmark automático com semáforos
  • Base para estudos de caso reais
